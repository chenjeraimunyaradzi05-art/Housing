# Typography

VÖR's typography system combines elegance with readability, using a carefully selected font pairing that reinforces our brand's sophisticated, professional identity.

## Font Families

### Playfair Display (Headings)
An elegant serif font that conveys sophistication and timelessness - perfect for a platform focused on building generational wealth.

### Inter (Body)
A highly legible sans-serif font designed for digital interfaces, ensuring excellent readability across all device sizes.

### JetBrains Mono (Code)
A developer-friendly monospace font for code snippets and technical information.

## Usage in Code

### CSS Variables

```css
/* Font families */
font-family: var(--font-heading);  /* Playfair Display */
font-family: var(--font-body);     /* Inter */
font-family: var(--font-mono);     /* JetBrains Mono */
```

### Tailwind Classes

```jsx
{/* Heading with Playfair Display */}
<h1 className="font-heading text-4xl font-semibold">
  Welcome to VÖR
</h1>

{/* Body text with Inter */}
<p className="font-body text-base text-gray-600">
  Start your investment journey today.
</p>

{/* Code with JetBrains Mono */}
<code className="font-mono text-sm bg-gray-100 px-2 py-1 rounded">
  npm install
</code>
```

## Best Practices

### Headings
- ✅ Use Playfair Display for all headings (h1-h6)
- ✅ Keep headings concise and meaningful
- ✅ Maintain proper heading hierarchy (don't skip levels)

### Body Text
- ✅ Use Inter for all body text, buttons, and UI elements
- ✅ Maintain comfortable line length (50-75 characters)
- ✅ Use appropriate line height for readability

### Emphasis
- Use **bold** for strong emphasis
- Use *italic* for titles and subtle emphasis
- Use `code` for technical terms and values

## Responsive Typography

| Tablet (640-1024px) | 2.5rem | 1rem |
