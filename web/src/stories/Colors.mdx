## Usage Guidelines

### Do's
- ✅ Use Rose (500) for primary CTAs and important interactive elements
- ✅ Use Lavender for secondary actions and hover states
- ✅ Use Teal for success messages and positive metrics
- ✅ Use Gold sparingly for premium features and highlights
- ✅ Ensure sufficient contrast (4.5:1 minimum for text)

### Don'ts
- ❌ Don't use multiple primary colors in the same section
- ❌ Don't use light colors (50-200) for text on white backgrounds
- ❌ Don't mix too many accent colors in one component
- ❌ Don't forget to test colors in both light and dark modes

## CSS Variables

All colors are available as CSS variables:

```css
/* Usage */
color: var(--color-rose-500);
background-color: var(--color-lavender-100);
border-color: var(--color-teal-600);
```

## Tailwind Classes

Use Tailwind utility classes for easy styling:

```jsx
<button className="bg-rose-500 hover:bg-rose-600 text-white">
  Primary Button
</button>

<span className="text-teal-600">
  +12.5% this year
</span>
```
